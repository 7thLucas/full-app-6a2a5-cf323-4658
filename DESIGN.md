# Design — SusieQ Kitchen
- Theme: pink-reddish. Primary ~#E0285A rose-red; soft pink background tints (#FFF0F3). Warm, food-friendly.
- Layout: single full-viewport hero, centered. Generous whitespace, minimal.
- Typography: bold modern display headline for name + tagline; clean sans body.
- CTA: one prominent rounded pink-red button with clear hover.
- Feel: fast-loading, airy, elegant, mobile-first. Avoid heavy imagery that slows load.
